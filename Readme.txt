User's Guide
- Open CLI
- Go to the directory where you downloaded the project
- type the following command <sv name_file>
- this proyect already contains an example that you can use type <sv names.txt>
- the program execution instructions appear once you enter this command
- Limitations:
    - Do not use filenames longer than 25 characters, and Do not use names in your HatoNames longer than 25 characteres also.
    - Is limiting to ASCII alphabet
